Docker is an open platform for developers and sysadmins to build, ship, and run distributed applications, whether on laptops, data center VMs, or the cloud.

This Bootcamp introduces Docker to an Absolute Beginner, This Bootcamp is followed by demos showing how to setup and get started with Docker. The coding exercises that accompany this course will help you practice Docker commands and developing your own images using Dockerfiles and practice Docker Compose. And finally, we have assignments to put your skills to the test. You will be given the challenge to solve using the skills you gained during this Bootcamp.

Agenda:-

Introduction
Docker Commands
Docker Run
Docker Images
Docker Compose
Docker Engine, Storage
Docker Networking
Docker Registry
Introduction of Container Orchestration - Docker Swarm & Kubernetes
Conclusion
